eCommerceMERN Stack

User-friendly, maintainable build

This project demonstrates a fully functioning eCommerce website, utilizing the MERN (MongoDB, Express, React, Node) stack.

Front end: ReactJS, HTML, CSS, Redux
Back end: NodeJS, ExpressJS, RESTful API
Database: MongoDB
Version Control: GIT BASH

Controlled user/admin management available through UI.
JWT authetication, user profile, effective cookie usage.

Display by category/brand/etc.
Sort by latest arrival/price/etc.
Maximize reusability with effective component use.
(ie: navigation menu, and category sidebar)

Admins may create, read, update and delete products/product information.

Images are uploaded automatically to the database.

No restarts or refreshes are needed to update rendering information.
All changes are reflected instantaneously in display.

Keeps profile of cart information until sign-out.
The user may delete, or change the quantity of items, as expected.

There are more features for demonstration yet to be shown.
